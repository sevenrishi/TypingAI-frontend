
# Typing AI – Complete Learning Curriculum

## PHASE 1 – KEYBOARD MASTERY

### Lesson 1 – Home Row Basics
**Keys:** A S D F   J K L ;
**Rule:** Fingers return to home row after every press.  
**Practice:**
```
asdf ;lkj
asdfg ;lkjh
```

---

### Lesson 2 – Upper Row
**Keys:** Q W E R T Y U I O P  
**Practice:**
```
qwert poiuy
qq ww ee rr tt yy uu ii oo pp
asdf qwer ;lkj poiuy
```

---

### Lesson 3 – Home Row Word Practice
**Words:**
all, fall, hall, ash, salsa, glass;'

**Paragraph:**
```
all fall hall ash salsa glass;'
```
